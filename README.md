# Go to TS

Small astro app using tree-sitter-web to convert go structs to typescript types.

Try it at [0xveya.github.io/go-to-ts](https://0xveya.github.io/go-to-ts/).

## Development

```sh
bun install
bun run dev
```

## Checks

```sh
bun run lint
bun run build
```
